IRC Client

You can use several optional fields to connect you on a server :

  • userName
  • realName
  • autoRejoin
  • autoConnect
  • secure
  • selfSigned
  • certExpired
  • floodProtection
  • floodProtectionDelay
  • sasl
  • stripColors
  • channelPrefixes
  • messageSplit

To use these fields you just need to place these names in a input ng-model attribute.

Example :

< input ng-model="realName" placeholder="enter your real name here" >
